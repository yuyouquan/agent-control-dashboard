# 🧠 子代理控制沙盘

可视化多代理系统控制面板，实时监控各个子代理的工作状态。

## 功能

- 🧠 可视化所有子代理
- 📊 显示每个代理状态（工作中/空闲/错误）
- 👆 点击查看详情（角色介绍、技能、工作内容）
- ➕ 创建新代理
- 🗑️ 删除代理
- 🔄 自动刷新（每30秒）

## 使用方法

1. 启动本地服务器:
```bash
python3 -m http.server 9999
```

2. 浏览器访问:
```
http://localhost:9999/agent-dashboard.html
```
